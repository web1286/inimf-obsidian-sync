这是一篇示例文章，用于测试 Obsidian 到 inimf.com 的同步功能。

## 正文格式

Obsidian 的文章会自动同步到 inimf.com 的「库存信息」模块。你可以在 inimf.com 后台将其分发到：

- 推荐文章
- 重要事件
- 高手身影
- 专题跟踪

## 图片支持

你可以使用以下两种方式插入图片：

1. **本地图片**：放在 `attachments/` 目录，然后引用 `![图片描述](attachments/图片名.png)`
2. **外链图片**：直接粘贴图片链接 `![图片描述](https://example.com/image.png)`

## 同步流程

1. 在 Obsidian 写文章
2. 保存后自动提交到 GitHub
3. GitHub Webhook 触发同步
4. 文章进入 inimf.com 库存信息
5. 在后台分发到各模块
